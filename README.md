### About ###
Demostrational Swift/UIKit app for viewing and playing local video files

#### Features: ####
   - List of videos to watch
   - Play selected video in player
   - All device orientations supported

#### Technology stack: ####
  - Swift/UIKit
  - Architecture MVC
  - Dependencies with CocoaPods

### Installation ####
```sh
   cd .../VideoPlayer
   pod install 
```
